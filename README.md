# 🍸 Velvet Pour — Interactive Cocktail Experience

A modern, animation-driven cocktail bar website built with **GSAP**, **React**, and **Tailwind CSS**, inspired by Awwwards-level design.

This project explores advanced web animation techniques like scroll-triggered storytelling, text splitting, and video scrubbing to create an immersive, cinematic user experience.

> 🎓 **Learning Resource:** This project was built by following and coding along with [**Master Web Animations in 2 Hours | Build an Awwwards-Level Website**](https://www.youtube.com/watch?v=AW1yfBKRMKc&t=203s) by **JavaScript Mastery**. The tutorial was an incredible resource for understanding GSAP from the ground up — from basic tweens to advanced scroll-driven animations. Highly recommended for anyone wanting to level up their animation skills.

## ✨ Features

### 🎬 Hero Experience

- Character-by-character text animation using **GSAP SplitText**
- Smooth staggered entrance animations
- Gradient-styled animated typography
- Scroll-synced video playback (scrubbing)
- Pinned hero section for cinematic storytelling

### 🌿 Scroll Interactions

- Parallax leaf animations using **ScrollTrigger**
- Smooth scroll-based transitions
- Responsive animation triggers for mobile & desktop

### 🍹 Cocktail Menu

- Dynamic rendering of cocktails & mocktails
- Clean, structured layout
- Subtle motion enhancements for better UX

## 🎯 Key Animation Concepts

| Concept | Description |
|---|---|
| `gsap.to()` / `gsap.from()` / `gsap.fromTo()` | Core tween methods for animating elements |
| `gsap.timeline()` | Sequencing multiple animations |
| `ScrollTrigger` | Pinning and scrubbing animations on scroll |
| `SplitText` | Breaking text into chars, words, and lines |
| Staggered animations | Cascading delays across multiple elements |
| Responsive animation logic | Adapting animations for different screen sizes |
